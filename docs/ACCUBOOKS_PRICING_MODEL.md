## Engineering Implementation Notes

### Feature Gating Strategy
```typescript
interface UserTier {
  tier: 'starter' | 'pro' | 'business' | 'enterprise';
  limits: {
    entities: number;
    users: number;
    transactions: number;
    apiCalls: number;
    aiQueries: number;
    workflows: number;
  };
  features: {
    aiCFO: boolean;
    forecasting: boolean;
    automation: boolean;
    payroll: boolean;
    multiEntity: boolean;
  };
}
```

### Usage Tracking Implementation
- Redis counters for real-time usage
- Database logs for monthly limits
- Middleware for API call counting
- Background jobs for limit enforcement

### Upgrade Flow
1. Soft limits with warnings at 80%
2. Hard blocks at 100%
3. Upgrade prompts with clear benefits
4. Seamless upgrade without data loss

### Rate Limiting
- API calls: Token bucket algorithm
- AI queries: Queue with priority tiers
- File uploads: Size and frequency limits
- Export jobs: Concurrent job limits

## ARR Maximization Strategy

### Pricing Psychology
- **Anchoring**: Show Enterprise price first to make Business seem reasonable
- **Decoy Effect**: Pro priced close to Business to push toward Business
- **Loss Aversion**: Highlight what users lose by downgrading
- **Social Proof**: Show what similar companies choose

### Expansion Revenue
- **User Seats**: $10/user/month beyond included limits
- **Add-on Modules**: Advanced analytics ($49/mo), Industry modules ($99/mo)
- **Usage Overage**: $0.10 per transaction beyond limits
- **Professional Services**: Implementation, training, custom reports

### Retention Mechanics
- **Data Gravity**: Historical data and customizations
- **Workflow Dependencies**: Business processes built in platform
- **AI Learning**: Models trained on company data
- **Integration Lock-in**: Connected systems and APIs

### Churn Prevention
- **Usage Monitoring**: Identify at-risk accounts early
- **Success Metrics**: Track time saved, errors caught, insights generated
- **Proactive Outreach**: Account managers for high-value customers
- **Flexible Options**: Temporary downgrades, payment plans for struggling customers
